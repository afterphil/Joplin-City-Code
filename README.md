\## Repository Structure



\* \*\*Root Folder:\*\*

&nbsp; Contains the official release of Joplin city codes in PDF format, sourced from \[Municode](https://www.municode.com/).



\* \*\*Child Directories (by Year):\*\*

&nbsp; Each subdirectory contains ordinances that were \*\*omitted from the official release\*\*, organized by year.



> Note: Some ordinances in these child directories may already be included in the official release, and some may still be missing despite best efforts.



---



\## Usage



1\. Access the \*\*official city codes PDF\*\* in the root folder for the official record.

2\. Explore child directories to review ordinances that were omitted or not captured in the official PDF.

3\. Use the folder structure to navigate ordinances by year.



This repository is intended for \*\*informational purposes\*\* and personal reference.
